# Financial Transaction Analytics Dashboard (2022–2024)

## 📊 Project Overview
This project analyzes financial transaction data using SQL and Power BI to uncover insights into customer behavior, revenue trends, and payment patterns.

## 🚀 Key Features
- Star Schema Data Model
- 9000+ Transactions analyzed
- 3 Years of data (2022–2024)
- Interactive Power BI Dashboard

## 📈 Key Insights
- Revenue shows consistent growth over 3 years
- Food and Shopping are top revenue-generating categories
- UPI is the most used payment method
- Weekend transactions show higher spending behavior

## 🛠️ Tools Used
- SQL (MySQL)
- Power BI
- Excel

## 📂 Project Structure
- `data/` → Dataset files
- `sql/` → Queries and schema
- `dashboard/` → Power BI file & screenshots
